Hacking & Development
2013-05-24 - Category: Readme - Tag: GettingStarted

Requirement

  • PHP 5.3+
  • Supported Phar extension

Clone Project & Setup

git://github.com/scarwu/Pointless.git
cd Pointless
./setup.sh

Set Environment

alias poid="/path/to/Pointless/src/Pointless.php"

Package PHP files

cd /path/to/Pointless
./build.php

Project files structure

.
├── bin                      - Pointless Phar file
└── src                      - Pointless source
    ├── Pointless.php        - Pointless execute shell
    ├── Boot.php             - Bootstrap loader
    ├── Command              - All Command's Implement
    ├── Library              - Pointless default library
    │   ├── Compress.php
    │   ├── ExtensionLoader.php
    │   ├── GeneralFunction.php
    │   ├── HTMLGenerator.php
    │   └── Resource.php
    ├── Plugin               - Git sub modules
    │   ├── Markdown
    │   └── NanoCLI
    └── Sample               - Under this folder's files will copy into user's blog
        ├── Config.php       - Blog configs
        ├── Extension        - Blog extension about sitemap, atom, and etc.
        ├── Markdown         - MD files
        └── Theme            - Blog themes
            └── Classic      - Theme name
                ├── Css      - CSS files
                ├── Js       - Javascript files
                ├── Script   - Convert recource and generte HTML files
                └── Template - HTML template

User blog files structure

.
├── Config.php - Blog configs
├── Deploy     - Git repository, to deploy blog
├── Extension  - Blog extension about sitemap, atom, and etc.
├── Markdown   - MD files
├── Public     - All HTML files
├── Resource   - Under this folder's files will copy into public folder
└── Theme      - Blog themes

If you want to know command usage detail, you can using this. eg, poi help switch.

Post Article
2013-05-24 - Category: Readme - Tag: GettingStarted

New article

Add new article to your blog.

poi add

This command will be create new article after Q&A. Default using vi to edit article, and you can changing it.

Edit article

poi edit

Delete article

poi delete

Last

If you want to add(edit/delete) static page that you can adding option -s behind the command. eg, poi add(edit/delete) -s.

Switch Blogs
2013-05-24 - Category: Readme - Tag: GettingStarted

Switch exists blog

If you have many blogs, you can using command switch to switch deference blogs.

poi switch <blog name>

Show all blogs

poi switch -l

Remove blog

This command will be remove blog record from the Pointless, but it doesn't delete any real files under the path.

poi switch -r <blog name>

First Usage
2013-05-24 - Category: Readme - Tag: GettingStarted

Initialize Blog

First, you have to initialize your blog folder. This folder will be store all your blog's files.

poi init <blog_name>

Configure

Second, Configuring your blog information.

poi config

Generate HTML

Third, generating your blog. This command will be create HTML files.
